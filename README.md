**EthioVerify** is a secure, lightweight mobile application designed to help businesses and individuals in Ethiopia instantly verify bank transactions. By combatting the rising issue of altered screenshots, EthioVerify bridges the trust gap between buyers and merchants during digital checkout.

---

## ✨ Key Features

* **📸 Smart Receipt Scanner:** Instantly scan physical or digital transaction receipts using OCR technology to parse data.
* **⌨️ Manual Verification:** Quick fallback to manually input reference/transaction IDs when a camera isn't viable.
* **🛡️ Fraud Prevention:** Cross-checks transaction details to eliminate the risk of fake or photoshopped payment screenshots.
* **⚡ Offline Validation Logic:** Built-in pattern verification algorithms run locally to flag invalid reference formats instantly without wasting data.
